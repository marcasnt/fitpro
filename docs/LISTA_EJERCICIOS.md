## NOTAS IMPORTANTES

### Formato de imágenes
- Formatos aceptados: JPG, PNG, WEBP, GIF (animado)
- Resolución recomendada: 600x400px mínimo
- Los GIFs animados muestran el movimiento completo del ejercicio

### Nombres de archivo
- Usar guiones medios (-) en lugar de espacios
- Usar solo minúsculas
- Incluir la extensión correcta (.jpg, .png, .gif, .webp)

### URL de Google Drive
Después de subir las imágenes, la URL pública será algo como:
```
https://drive.google.com/file/d/1ABC123xyz/preview
```

Para usar en la hoja de cálculo, convertir a formato de embed:
```
https://drive.google.com/uc?export=view&id=1ABC123xyz
```
